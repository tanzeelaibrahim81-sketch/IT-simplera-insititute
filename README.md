Task Management System

A modern Full Stack Task Management System built as part of a Full Stack Web Development Internship assignment. The application allows users to register, log in securely, and manage their daily tasks through a clean and responsive interface.

Features

- User Registration
- User Login
- JWT Authentication
- Password Hashing using bcrypt
- Protected Routes
- Create, Read, Update and Delete (CRUD) Tasks
- Responsive Dashboard
- User Profile
- Form Validation
- Error Handling
- Loading Indicators
- Toast Notifications
- Mobile-Friendly Design
- Dark/Light Theme

Tech Stack

Frontend

- React.js
- HTML5
- CSS3
- JavaScript (ES6+)
- Axios

Backend

- Node.js
- Express.js

Database

- MongoDB
- MongoDB Compass

Authentication

- JSON Web Token (JWT)
- bcrypt

Installation

Clone the repository

git clone <your-github-repository-link>

Install dependencies

Backend

cd server
npm install

Frontend

cd client
npm install

Configure Environment Variables

Create a ".env" file inside the "server" folder and add:

PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key

Run the Project

Backend

npm run dev

Frontend

npm start
